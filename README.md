RugRadar is a real-time token risk scanner built for crypto degens. 
Paste any contract address and get an instant Rug Probability Score 
based on live liquidity, market cap, sell pressure, token age, and 
contract verification data pulled directly from DexScreener and Etherscan.

Features:
- Real-time rug score (0-100%)
- Live liquidity, volume, and price data
- Contract verification checks
- Risk flags with plain English explanations
- Hall of Shame — confirmed historical rugs
- Dark, Light, and Hacker themes
- One-click tweet generator

Not financial advice. DYOR. But also trust the data.
